# Expense Management System - User Guide

## Getting Started

### For Employees

#### Creating an Expense
1. Navigate to the Expenses page
2. Click "Add Expense" button
3. Enter the amount in GBP
4. Select the expense date
5. Choose the appropriate category
6. Add a clear description
7. Click "Submit" to save as draft

#### Viewing Your Expenses
- The main Expenses page shows all your expenses
- Use the Filter box to search by category, status, or description
- Click on any expense to view details

### For Managers

#### Approving Expenses
1. Navigate to "Approve Expenses"
2. Review the list of pending (submitted) expenses
3. Select the expenses you want to approve
4. Click "Approve" button
5. Expenses are automatically processed for payment

#### Using the Filter
- Type keywords to find specific expenses
- Filter by category name, amount, or employee name
- Clear filter to see all pending expenses

## Tips & Best Practices

### For Better Experience
- Submit expenses promptly (within 30 days)
- Always include detailed descriptions
- Keep digital copies of receipts
- Check expense status regularly

### Common Mistakes to Avoid
- ❌ Submitting duplicate expenses
- ❌ Missing expense descriptions
- ❌ Claiming personal expenses
- ❌ Exceeding category limits without approval

## Features

### Natural Language Chat (Optional)
If your organization has enabled the GenAI Chat UI, you can:
- Ask questions about expenses in natural language
- Create expenses by describing them
- Get summaries and reports
- Find specific expenses quickly

Example queries:
- "Show me all my travel expenses from last month"
- "Create a £45 expense for a client lunch"
- "What's the total of my approved expenses?"
- "List all pending expenses awaiting approval"

### API Access
Developers can use the REST API to integrate with other systems:
- Full CRUD operations on expenses
- Swagger documentation at `/swagger`
- Authentication via Azure AD
- JSON response format
